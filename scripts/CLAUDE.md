# CLAUDE.md - Scripts Package

## Package: Scripts

### Python Scripts Requirements

- Python 3.8+
- Type hints required
- Docstrings for all functions
- Error handling with proper logging

### Node Scripts Requirements

- ESM modules (.mjs extension)
- Use commander for CLI interfaces
- Proper error handling and exit codes
- Help text for all commands

### Common Scripts

- `fix_imports.py` - Auto-fix import statements
- `monitor.py` - System monitoring and health checks
- `validate_system.py` - Validate project structure and dependencies

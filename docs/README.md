# Documentation Directory

This directory contains all project documentation organized by category.

## Structure

- **`project/`** - Project management and development docs
  - `CLAUDE.md` - AI assistant instructions and conventions
  - `context_recovery.md` - Current project state and session handoffs
  - `todo_list.md` - Task tracking and progress
  - `ARCHITECTURE.md` - Technical patterns and decisions
  - `WORKFLOW.md` - Development workflow
  - `git-branching-protocol.md` - Git conventions

- **`progress/`** - Individual task tracking and execution plans
  - Detailed task files with context for fresh instances
  - Execution plans and progress updates
  - Collaborative task notes

- **`design/`** - Visual design and branding
  - `style-guide.md` - Comprehensive style guide
  - `BRANDING-GUIDE.md` - Brand identity guidelines

- **`testing/`** - Testing documentation
  - `testing-strategy.md` - Testing approach and gaps
  - `TESTING_SUMMARY.md` - Current test results
  - `components/` - Component-specific test docs
  - `migrations/` - Migration test results

- **`migration/`** - Historical migration docs
  - Phase documentation
  - Migration summaries
